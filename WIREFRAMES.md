## Design System Specifications

### Color Palette
```
Primary Colors:
- Coffee Brown:    #6F4E37
- Cream:           #F5F5DC
- Dark Roast:      #3E2723

Secondary Colors:
- Caramel:         #C68E5B
- Foam White:      #FAFAFA
- Steam Gray:      #9E9E9E

Accent Colors:
- Gold (Rewards):  #FFD700
- Green (Success): #4CAF50
- Red (Alert):     #F44336

Text Colors:
- Primary:         #212121
- Secondary:       #757575
- Disabled:        #BDBDBD
```

### Typography
```
Font Family: Inter, SF Pro (iOS), Roboto (Android)

Headings:
- H1: 32px, Bold
- H2: 24px, Semibold
- H3: 20px, Semibold
- H4: 18px, Medium

Body:
- Large:   16px, Regular
- Medium:  14px, Regular
- Small:   12px, Regular

Button Text: 16px, Semibold
Caption: 12px, Regular
```

### Spacing System
```
Base Unit: 8px

Spacing Scale:
- XXS: 4px   (0.5x)
- XS:  8px   (1x)
- S:   16px  (2x)
- M:   24px  (3x)
- L:   32px  (4x)
- XL:  48px  (6x)
- XXL: 64px  (8x)

Component Padding: 16px
Card Spacing: 12px
Section Margin: 24px
```

### Component Library

#### Buttons
```
Primary Button:
- Background: Coffee Brown (#6F4E37)

- Height: 48px
- Border Radius: 8px

Secondary Button:
- Background: Transparent
- Border: 1px solid Coffee Brown
- Text: Coffee Brown
- Height: 48px
- Border Radius: 8px

Icon Button:
- Size: 44x44px
- Background: Transparent
- Icon: 24x24px
```

#### Cards
```
Standard Card:
- Background: White
- Border Radius: 12px
- Shadow: 0 2px 8px rgba(0,0,0,0.08)
- Padding: 16px

Product Card:
- Image: 16:9 ratio
- Padding: 12px
- Hover: Slight lift effect
```

#### Input Fields
```
Text Input:
- Height: 48px
- Border: 1px solid #E0E0E0
- Border Radius: 8px
- Padding: 12px 16px
- Focus: 2px border, Coffee Brown
```

## Animation & Interactions

### Transitions
```
Screen Transitions: 300ms ease-in-out
Button Press: Scale 0.95, 100ms
Card Tap: Subtle lift, 200ms
Loading: Skeleton screens / Spinner
Pull to Refresh: 500ms spring
```

### Microinteractions
```
Add to Cart: 
- Item flies to cart icon
- Cart badge animates

Order Status:
- Progress bar fills smoothly
- Status icons animate in sequence

Rewards:
- Stars sparkle when earned
- Level-up celebration animation

Payment Success:
- Checkmark animation
- Success message fade-in
```
